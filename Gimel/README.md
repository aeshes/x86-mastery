# Подробнее о байте ModR/M

| Значение Mod | Режим адресации                  | Пример           | Дополнительные сведения
| ------------ |:---------------------------------|------------------|-----------------
| **11b**      | Регистровая адресация            | mov eax, ecx     | В R/M находится номер регистра
| **00b**      | Регистровая косвенная адресация  | mov eax, [ecx]   |
